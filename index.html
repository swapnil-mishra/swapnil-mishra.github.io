Portfolio

Most recent works

Sentidatalyzer Application  

A sentiment analysis app that uses predictive analysis crucial for business growth, offering precise sentiment insights from customer feedback. It informs decision-making, helps tailor strategies, products, and services, enhancing satisfaction and loyalty. Additionally, it predicts trends, keeping businesses ahead and enabling proactive adjustments.
Technologies used: Languages:Python jinjas 2 for creating html templates; Framework: Flask; Libraries: Numpy, Pandas, Seaborn, Matplotlib, Natural Language Toolkit (NLTK), Scikit-learn; Data visualisation: Matplotlib, Seaborn and Word Cloud.

BookLoop-An Android Book Recycle App  

An Android app designed to facilitate the exchange of pre-owned books, connecting both buyers and sellers. This book recycling application offers users the ability to browse listings of used books from other sellers and sell their own used books to registered buyers within the app. It was developed using Java, Android Studio and Firebase technologies.

E-Vidyut Training Portal  

A user-friendly Learning Management System (LMS) portal that allows students to easily register and access course content. The front-end of the portal was developed using HTML, CSS, and JavaScript, while PHP was employed for the back-end functionality. To manage the database connection, MySql was utilized.

Shopping Cart Order Summary  

The shopping cart's order summary is powered by React.js and structured into five sections:
Main Layer: Contains cart products sourced from JSON data through product mapping.
Header Layer: Houses the "order summary" navigation bar.
Cart Layer: Manages added and modified cart items.
Basket Layer: Computes the total cost after applying discounts.
Two-Column Layout: Dynamically updates item details and costs as items and quantities change.
